## What AgentHub is, in one sentence

AgentHub is a **self-hosted web app** that boots a containerized Linux workspace with a coding agent (Claude Code, OpenCode, MiniMax, or Droid) pre-installed, so you can drive that agent from a browser terminal and let it deploy its own apps through the bundled `agentdeploy` MCP.

Unlike hosted products, every workspace runs on hardware you own. No API keys leave your network unless you put them there yourself.
