Trust the AgentHub CA

AgentHub generated a private certificate authority for this install. Import it on each device that connects to AgentHub from this LAN — once per device. Then your browser will trust the AgentHub HTTPS certificate just like any public site.

Download CA certificate

macOS

  1. Click the download button above.
  2. Run this in Terminal:
sudo security add-trusted-cert -d -r trustRoot \
  -k /Library/Keychains/System.keychain ~/Downloads/agenthub-ca.crt

You'll be prompted for your macOS user password. Restart your browser to pick up the new trust root.

iOS

  1. Tap the download button above on your iOS device.
  2. Open Settings → General → VPN & Device Management.
  3. Under "Downloaded Profile", tap the AgentHub CA, then tap Install.
  4. Open Settings → General → About → Certificate Trust Settings.
  5. Toggle ON the trust switch next to AgentHub Self-CA.

Android

  1. Download the certificate to your device.
  2. Open Settings → Security → Encryption & credentials → Install a certificate → CA certificate.
  3. Select the downloaded file and confirm.

Android distinguishes user-installed CAs (apps may not trust them) from system CAs. For most browsers, user-installed is sufficient.

Linux (Debian/Ubuntu)

sudo cp ~/Downloads/agenthub-ca.crt /usr/local/share/ca-certificates/agenthub.crt
sudo update-ca-certificates

Firefox (separate trust store)

  1. Open about:preferences#privacy.
  2. Scroll to Certificates → View Certificates.
  3. Switch to the Authorities tab and click Import….
  4. Select agenthub-ca.crt and check Trust this CA to identify websites.

Windows

  1. Download the certificate and double-click it.
  2. Click Install Certificate… → Local Machine → Next.
  3. Select Place all certificates in the following store → Browse → Trusted Root Certification Authorities.
  4. Click OK, Next, Finish.
